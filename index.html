    About

    MBP is program writed by HafisCZ. This program serves as a backup manager, primarily for Minecraft. It includes the ability to backup any folder to a .zip file, restore the folder from a backup and easy backup management.

    You you have any tips & ideas, or you want to report error, send me an e-mail.

    Important: Actually, you need to run program with following argument, if You want to use cloud service. -Djava.net.preferIPv4Stack=true

    Features

    • Manual backup creation & use
    • Backup management, Rename & Remove
    • Basic FTP interface (Upload, Download and Remove)
    • Automatic backup creation with split modified by user (Max. 23 hours & 59 minutes)

    WIP Features

    • Internal command line / output
    • FTP interface

    Planned features

    • Integrated access to user's cloud (Google Drive, DropBox or via FTP transfer)

    Requirements

    • Java 7 or newer
    • Any current Operating System (Does not support Windows XP and older systems)
