    About

    MBP is program writed by HafisCZ. This program serves as a backup manager, primarily for Minecraft. It includes the ability to backup any folder to a .zip file, restore the folder from a backup and backup management.

    You you have any tips & ideas, or you want to report error, send me an e-mail.

    Features

    • Manual backup creation & use
    • Backup management, Rename & Remove
    • Automatic backup creation with split modified by user (Max. 23 hours & 59 minutes)

    WIP Features

    • FTP Upload & Download

    Planned features

    • Integrated access to user's cloud (Google Drive, DropBox or via FTP transfer).
