    MBP is program writed by HafisCZ. This program serves as a backup manager, primarily for Minecraft. It includes the ability to backup any folder to a .zip file, restore the folder from a backup and easy backup management. Screenshots are at the end of the page.

    If you have any tips & ideas, or you just want to report error, send me an e-mail.

    Important: Actually, you need to run program with following argument, if You want to use cloud service. -Djava.net.preferIPv4Stack=true

    Features

    • Manual backup creation & use
    • Backup management, Rename & Remove
    • Basic FTP interface (Upload, Download and Remove)
    • Timed manual backup creation with split modified by user (Max. 23 hours & 59 minutes)
      When time runs out waits for confirmation to create backup
    • Local file description - Add & Modify & Remove in one button
    • Window with text reports

    WIP Features

    Application settings

    • This app is using 2 files (data.ini and cloud.ini) for storing all settings

    • data.ini's file description:

      [dateformat]
      0 // Choosen date and time format, ranging from 0 to 5
      [path]
      C:\savefolder // Path to your map folder which will be stored in backup file
      C:\backup // Path to folder where You want to store your backup files
      C:\mc.exe // Path to your Minecraft file
      [timebackup]
      0 // Boolean value, enable or disable timed saving
      36611 // Period, after which will be map automatically backed if timed backup is started

    • cloud.ini's file description:

      [cloud]
      0 // Boolean value, enable or disable FTPService
      127.0.0.1 // IP used by FTPService, IP of desired FTP server
      21 // Port which will be used for connecting to FTP server
      USER // Username used for connecting to FTP server
      USER // Password used for connecting to FTP server
